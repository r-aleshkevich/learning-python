# My Python Journey 🚀

Репозиторий для хранения учебных проектов, практики и подготовки к поступлению на ФПМИ БГУ (специальность «Программная инженерия»). Цель — развитие в сфере Machine Learning (ML).

## 📁 Структура проекта:
* укус питона — выполнение практических заданий и упражнений по книге «A Byte of Python» [укус питона].
* игры — консольные проекты и первые игры на Python (включая игру «Цуефа»).
* практика — разбор базовых тем синтаксиса, алгоритмов и подготовка к изучению ООП.

## 🛠️ Стек технологий:
* Python 3
* Git / GitHub

*План на 11 класс: успешно сдать экзамены, закрепить ООП и начать изучение библиотек NumPy и Pandas.*
